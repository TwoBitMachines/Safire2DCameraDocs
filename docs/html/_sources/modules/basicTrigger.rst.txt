Basic Trigger
+++++++++++++

The basic trigger can be used for any miscellaneous events that require
the OnEnter() and OnExit() Unity Events.

.. list-table::
   :widths: 25 100
   :header-rows: 1

   * - Property
     - 

   * - Trigger
     - Set to Trigger Once if you want this trigger to only activate once. 
       With Trigger, the target can activate the trigger any number of times.

   * - On Enter
     - The Unity Event invoked when the target enters the trigger.

   * - On Exit
     - The Unity Event invoked when the target exits the trigger.
